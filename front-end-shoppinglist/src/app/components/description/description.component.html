Um den Nutzer zu Informieren habe ich verschieden Wege gewählt, dies wirkt an manchen Stellen inkonsistent. Dies habe ich bewusst getan, um die verschiedenen Möglichkeiten zu veranschaulichen.

Navigation

In the header you can access all areas of the application (as long as you are logged in).

If the diplay is too narrow (mobile usage), there is an alternative menu bar on the left side. This can be folded out via the menu icon on the top left.

Log In

Log in with your personal Mail und Password

Important is to log in an account you had been createt before!

New Account

Create an account. At least one character must be entered everywhere.

In addition, the e-mail must be in the form of an e-mail and the password must have one of the following components:

0-9, A-Z, a-z, !'$%^&*".

List Administration

In the upper area, lists can be created by entering a name at (List Name*) and creating the list with Create List.

Important: each list name is unique and may therefore only be assigned once!

Below you will find a selection of lists that may have been created. The checkbox on the right side must be activated in order to be able to use the control fields further down.


Users can be added to any self-created list. These users then see the list and its entries in "Lists", e.g. to buy entries themselves and then mark them as purchased.

Adding only works via an e-mail address stored in the system.


To use the "Choose mail to Remove" drop-down menu, you must first select a list with the corresponding user.

After that, the mail addresses of the users are available and can be removed from the respective list by clicking on the remove button.

Lists (Mobile usage - small screen)

Important: there are two completely different views. Desktop variant can be selected via the hadder menu with wide screen via "Lists".

In the upper left corner you can select a list. You can choose between lists that you have created yourself and lists that other users have added to.

Next to it is a selection menu to explicitly set the first category of the list.

After a list has been selected, items can be marked as purchased or deleted (by the list creator). Deleting in mobile mode (with narrow display) is done by long press on the element. A popup appears to confirm the deletion.

Marking as "bought by" is done by clicking on the slider in the right part of the entries. Adding can be found on the right. Product* (name) is freely selectable and should be completed with a quantity and then a product category must be selected.

Lists (Desktop Version - wide Screen)

Important: there are two completely different views. The mobile version can be selected via the page navigation on the left side of the screen by clicking on "Lists".

In the upper left corner you can select a list. You can choose between lists that you have created yourself and lists that other users have added to.

Next to it is a selection menu to explicitly set the first category of the list.

After a list has been selected, an overview of the individual product categories appears on the left. If you select a category, the articles appear to the right as an icon and can be deleted there (by the creator of the list). Deleting in desktop mode (with wide display) is done by clicking on the X in the icon.

Marking as purchased is done by clicking on the respective product. Adding can be found on the right. Product* (name) is freely selectable and should be completed with a quantity and the product category is the currently selected cathegory./p>

personalized button with your initials always on the top right corner

When you activate it, a submenu appears. In this menu you can log out or delete your account, including all your lists and their entries.

Navigation

Im Header können sie (soweit sie eingelogt sind, auf alle bereiche der Applikation zugreifen.

Sollte das Diplay zu schmal werden (mobile Nutzung) gibt es eine alternative Menüleiste am linken Rand. Diese kann über das Menü-Symbol Links Oben ausgeklappt werden.

Log In

Loggen Sie sich mit Ihrem persönlichen Mail und Passwort ein

Wichtig ist, dass Sie sich mit einem Konto anmelden, das Sie zuvor erstellt haben.

New Account

Erstellen sie sich einen Account. Es muss überall mindestens ein Zeichen eingegeben werden.

Zusätzlich muss die E-Mail der Form einer E-mail entsprechen und das Passwort muss jeweils eines der folgenden Bestandteile aufweisen:

0-9, A-Z, a-z, !'$%^&*"

List Administration

Im oberen Bereich können Listen erstellt werden, indem ein Name bei (List Name*) hinterlegt wird und mit Create List die Liste angelegt wird.

Wichtig: jeder Listenname ist eindeutig und darf daher nur ein mal vergeben werden!

Darunter befindet sich eine Auswahl an ggf. erstelten Listen. Die Checkbox auf der rechten Seite muss aktiviert werden, um die Bedienfelder weiter unten benutzen zu können.


Zu jeder selbst erstellten Liste können User hinzugefügt werden. Diese User sehen dann die Liste und deren Einträge in "Lists" um z.B. Einträge selbst zu kaufen und diese dann als gekauft zu markieren.

Das Hinzufügen funktioniert nur über eine im System hinterlegte E-Mail Adresse.


Um das Dropdown Menü "Choose mail to Remove" zu benutzen, muss zurest eine Liste mit dem entsprechenden User ausgeählt werden.

Danach stehen die Mailadressen der Nutzer zur Verfügung und können mit einem klick auf den remove Knopf aus der jeweiligen Liste entfernt werden.

Lists (mobile Version - schmaler Bildschirm)

Wichtig: es gibt zwei völlig verschiedene Ansichten. Desktopvariante kann über das Haddermenue bei breitem Bildschirm via "Lists" angewaehlt werden.

In der Oberen linken Ecke kann eine Liste Ausgewählt werden. Zur Auswahl stehen Listen, die selbst erstellt wurden, und Listen zu denen andere Benutzer einen hinzugefügt haben.

Daneben ist ein Auswahlmenue um die erste Kategorie der Liste Explizit festzulegen.

Nachdem eine Liste Gewählt wurde erscheint links eine Uebersicht der einzelnen Prduktkategorien. Wählt man eine Kathegorie aus, erscheinen rechts daneben die Artikel als Icon und können dort (vom Ersteller der Liste) auch gelöscht werden. Löschen im Desktop Modus (mit breitem Display) erfolgt über klicken auf das X im Icon.

Das Markieren als "gekauft durch" wird durch klicken auf den Slider im rechten Teil der Einträge getätigt. Das Hinzufügen ist unten zu finden. Product* (Name) ist frei wählbar und sollte mit einer Menge ergänzt werden und dann muss eine Produktkategorie gewählt werden.

Lists (Desktop Version - breiter Bildschirm)

Wichtig: es gibt zwei völlig verschiedene Ansichten. Die mobile Version kann bei schmalem Bildschirm ueber die Seiten-Navigation am linken Bildschirmrand durch Klicken auf "Lists" gewählt werden.

In der Oberen linken Ecke kann eine Liste Ausgewählt werden. Zur Auswahl stehen Listen, die selbst erstellt wurden, und Listen zu denen andere Benutzer einen hinzugefügt haben.

Daneben ist ein Auswahlmenue um die erste Kategorie der Liste Explizit festzulegen.

Nachdem eine Liste Gewählt wurde können dort Artikel als gekauft markiert werden oder (vom Ersteller der Liste) auch gelöscht werden. Löschen im mobilen Modus (mit schmalem Display) erfolgt über langes drücken auf das Element. Daraufhin erscheint ein PopUP zum bestätigen.

Das Markieren als gekauft wird durch klicken auf das jeweilige Produkt erledigt. Das Hinzufügen ist rechts zu finden. Product* (Name) ist frei wählbar und sollte mit einer Menge ergänzt werden und die Produktkategorie ist die aktuell gewählte Kathegorie.

personalisierter Knopf Ihren Initialien, immer oben linke Ecke
